# NLP Preprocessing & Word Embedding App

This is a Streamlit-based NLP web application that performs:

- Tokenization
- Text Cleaning
- Stemming
- Lemmatization
- Bag of Words
- Word2Vec Embeddings

## Tech Stack
Python, Streamlit, NLTK, spaCy, Scikit-learn, Gensim

## How to Run
pip install -r requirements.txt
streamlit run app.py
